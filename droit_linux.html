Voir, modifier et attribuer les permissions

1. Catégories et permissions

Chaque fichier/dossier possède 3 catégories :

  • Owner (u) : propriétaire
  • Group (g) : groupe associé
  • Others (o) : autres utilisateurs

Et 3 types de permissions :

  • r : lecture
  • w : écriture
  • x : exécution
Exemple : -rwxr-xr--
Owner : rwx
Group : r-x
Others: r--

2. Voir les droits

# Affiche les droits des fichiers dans le répertoire courant
ls -l

# Affiche droits et attributs étendus
ls -la

3. Modifier les droits avec chmod

Deux méthodes : notation lettre ou numérique

3.1 Notation lettre

# Ajouter l'exécution pour le propriétaire
chmod u+x fichier.txt

# Retirer l'écriture pour le groupe
chmod g-w fichier.txt

# Ajouter lecture pour tous
chmod a+r fichier.txt

# u = owner, g = group, o = others, a = all

3.2 Notation numérique (octale)

Octal Lettres Permission
0---Aucun droit
1--xExécution
2-w-Écriture
3-wxÉcriture + exécution
4r--Lecture
5r-xLecture + exécution
6rw-Lecture + écriture
7rwxLecture + écriture + exécution
 
# Exemple : chmod 750 fichier.txt
Owner : 7 → rwx
Group : 5 → r-x
Others: 0 → ---

4. Modifier propriétaire et groupe

# Changer le propriétaire
sudo chown thomas fichier.txt

# Changer le groupe
sudo chgrp admins fichier.txt

# Changer propriétaire et groupe en même temps
sudo chown thomas:admins fichier.txt

5. Commandes utiles pour l'administration

# Créer un nouveau groupe
sudo groupadd devs

# Ajouter un utilisateur à un groupe
sudo usermod -aG devs thomas

# Supprimer un utilisateur
sudo deluser thomas

# Vérifier les utilisateurs
cat /etc/passwd

# Vérifier les groupes
cat /etc/group

6. Exemples pratiques

# Fichier exécutable par propriétaire, lecture pour groupe, aucun droit pour autres
chmod 740 script.sh

# Répertoire accessible en lecture et exécution pour tous, écriture pour owner seulement
chmod 755 /var/www/html

# Fichier texte lisible par tous, modifiable seulement par owner
chmod 644 notes.txt
