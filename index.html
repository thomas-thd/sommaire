Situations Professionnelles & Compétences

Suivi du parc informatique

Contexte : Parc Windows pour utilisateurs INTM

Action : Inventaire des postes via Intune et vérification manuelle sur site.

Résultat : Base de données fiable pour planifier mises à jour et migrations sans interruption.

Compétences : Gestion de parc, Intune, inventaire matériel.

Réseau et infrastructure physique

Contexte : Cœur de réseau situé à proximité d’un site sensible (Ministère de l’Intérieur), avec perturbations du signal mobile liées à un brouillage GSM.

Objectif : Maintenir une connectivité de secours fiable malgré un environnement radio contraint.

action : Installation physique de bornes GSM 4G et mise en place du paramétrage réseau nécessaire (adressage, intégration au LAN, tests de connectivité).

Résultat : Mise en service d’une solution de secours fonctionnelle permettant d’assurer la continuité du réseau.

Structuration de Données Techniques (JSON vers Excel)

Contexte : Besoin d'extraire des informations d'infrastructure contenues dans des plans techniques au format DWG.

Objectif : Rendre les données exploitables par les services de gestion pour un suivi précis du patrimoine technique.

Action : Extraction des données structurées en JSON depuis les fichiers sources et création d'un script ou d'un processus de conversion vers Microsoft Excel.

Résultat : fiabilisation de la base de données technique.

Compétences : Manipulation de fichiers JSON, Data management, Automatisation.
