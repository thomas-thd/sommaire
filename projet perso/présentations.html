Présentation des logiciels

Découvrez les outils et technologies que j'utilise pour le développement web et la création d'interfaces utilisateur.

HTML

HTML (HyperText Markup Language) est le langage standard utilisé pour structurer les pages web. C'est la base de tout site internet.

CSS

CSS (Cascading Style Sheets) permet de styliser et de mettre en forme les pages HTML en définissant les couleurs, les polices, les marges, et plus encore.

JavaScript

JavaScript est un langage de programmation utilisé pour ajouter de l'interactivité aux pages web. Il permet de créer des éléments dynamiques comme des animations ou des formulaires interactifs.

React

React est une bibliothèque JavaScript populaire pour la création d'interfaces utilisateur. Il permet de créer des applications web rapides et évolutives.

Vue.js

Vue.js est un framework JavaScript utilisé pour développer des interfaces utilisateurs modernes. Il est facile à intégrer et à utiliser.

Node.js

Node.js est un environnement d'exécution JavaScript côté serveur. Il permet de créer des applications backend performantes et scalables.

PHP

PHP est un langage de programmation côté serveur utilisé pour créer des sites dynamiques et gérer des bases de données, souvent en tandem avec MySQL.
